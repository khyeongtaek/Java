자바 개발 방법

1. 자바 소스 코드를 작성합니다.

2. 저장할 때 .java를 사용합니다.

3. 자바의 모든 소스 코드는 클래스 내부에 작성합니다.

4. 파일 하나당 하나의 클래스만 작성하는 것을 권장합니다.

5. 저장할 때 파일명은 클래스명과 동일한 이름을 사용합니다.

6. HelloWorld.java 파일이 있는 경우 가장 먼저 컴파일을 합니다. (javac.exe 파일 담당)
  (컴파일 : 문자로 작성한 소스 코드 -> 이진 데이터로 바꾸는 작업)
  (인간이 작성한 문자를 기계가 해석할 수 있는 기계어로 변환하는 작업)
  * 컴파일 오류 : 소스 코드가 틀렸다. 문법이 안 맞는다. (이클립스가 잡아준다.)

7. 컴파일 성공 시 HelloWorld.class 파일이 새로 생성됩니다.

8. HelloWorld.class 파일을 실행합니다. (java.exe 파일 담당)
